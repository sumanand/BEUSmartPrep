# BEU SmartPrep (Android App)

> Smart Study Companion for Bihar Engineering University (BEU) Students 🚀

BEU SmartPrep is an Android application built using Java, designed to help BTech students under BEU access study materials, notes, and previous year questions in a structured and easy way.

---

## 📌 About the App

BEU SmartPrep provides a centralized platform for Bihar Engineering Students who is pusuing BTech under BEU(Bihar Engineering University) where students can find all their academic resources without searching multiple sources.

🎯 The app focuses on simplicity, speed, and usability.

---

## 🚀 Features

### 📚 Study Materials

* Semester-wise Notes
* Subject-wise Content
* Previous Year Questions (PYQs)
* Important Topics

### 🔍 Smart Functionalities

* Search Study Materials
* Filter by Semester & Subject
* Easy Navigation

### 🔐 User System

* Secure Login & Signup
* Firebase Authentication

### 👨‍💻 Admin Features

* Upload Notes & PDFs
* Manage Content
* Real-time Updates

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash id="n9x6kp"
git clone https://github.com/sumanand/beusmartprep.git
```

### 2️⃣ Open in Android Studio

* Open Android Studio
* Click on **Open Project**
* Select the cloned folder

### 3️⃣ Sync Gradle

* Let Gradle sync automatically
* Install required SDK if prompted

### 4️⃣ Run App

* Connect Android device / Emulator
* Click ▶ Run

---

## 🔥 Firebase Setup

1. Go to Firebase Console
2. Create a new project
3. Add Android App
4. Download **google-services.json**
5. Paste inside:

```id="k7o5nq"
app/google-services.json
```

6. Enable:

* Authentication
* Realtime Database / Firestore
